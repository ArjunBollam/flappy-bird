# Flappy Bird.io on Vercel

This is a static deployment of the Flappy Bird game. All files are served from the root. No backend is required.

## Deploying to Vercel
1. Push this repo to GitHub.
2. Import the repo in Vercel and deploy as a static site.
3. No build step is needed.

## Notes
- All assets are in the repo and referenced with root-relative paths.
- If you add new files, push to GitHub and redeploy.
